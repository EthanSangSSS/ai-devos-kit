# Dev Workflow SOP

idea -> create project kit -> local preflight -> first commit -> GitHub repo -> PR -> ChatGPT GitHub Connector review -> Codex local fix -> tests -> Computer Use QA when requested -> TestFlight -> App Store submission -> post-release monitoring -> next version backlog.

每阶段必须记录 branch、HEAD、diff stat、commands、pass/fail、remaining risk、NEXT_CODEX_PROMPT。
