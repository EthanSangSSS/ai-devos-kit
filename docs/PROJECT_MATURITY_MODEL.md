# Project Maturity Model

## L0 idea
Required: goal note, risk sketch. Next: template install.

## L1 prototype
Required: repo, AGENTS.md, README, first commit. Tests: smoke. Audit: secret scan.

## L2 usable demo
Required: architecture, testing doc, basic workflows. Tests: unit/smoke. Next: beta risks.

## L3 testable beta
Required: test matrix, bug cards, CI. Tests: automated + simulator. Audit: P0/P1 closed.

## L4 TestFlight ready
Required: privacy/IAP/screenshots/review notes. Tests: TestFlight smoke. Audit: App Store checklist.

## L5 App Store ready
Required: archive, metadata, rejection response plan. Tests: full regression. Audit: security + IAP + release.

## L6 post-release optimized
Required: monitoring, backlog, support loop. Tests: regression per release. Audit: incident and churn learning.
