# Codex Budget Maximization SOP

- ChatGPT: upfront audit, prompt generation, PR review, strategy.
- Codex: local execution, build, test, bug fix, commit.
- Claude Code: large UI/refactor with diff and test constraints.
- Gemini/Agy: long-context scan, low-cost draft, batch review.
- Local MLX: offline draft, log summary, non-authoritative notes.

Every Codex round starts with preflight and ends with NEXT_CODEX_PROMPT.
