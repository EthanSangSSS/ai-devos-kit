# OSS Readiness Notes

`ai-devos-kit` is intended to complement `web-chatgpt-skills`:

- `web-chatgpt-skills` packages reusable skill instructions for Web ChatGPT, Codex, and local-agent workflows.
- `ai-devos-kit` packages local-first scripts, prompts, templates, SOPs, and handoff contracts for running those workflows safely in repositories.

## Current readiness status

- Public README: present.
- License: MIT.
- Contribution guide: present.
- Security policy: present.
- PR template: present.
- CODEOWNERS: present.
- Public-safety validator: present.
- CI public-safety workflow: present.
- Release notes: prepared.

## Review boundary

This repository should not be evaluated on unverified stars, downloads, production adoption, or external users. Its value is workflow discipline: version-sync gates, safe handoffs, reviewable run artifacts, and public-safe automation templates.

## Relationship to private downstream projects

This public kit may be used by private downstream repositories, but private downstream source, logs, secrets, customer data, employer data, or local runtime artifacts must not be copied into this repository.
