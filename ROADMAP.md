# Roadmap

This roadmap keeps `ai-devos-kit` focused on local-first, evidence-bounded AI-agent development workflows.

## v0.1.x — Public workflow baseline

- Keep README, license, contribution guide, security policy, PR template, and CODEOWNERS current.
- Maintain a dependency-free public-safety scan for docs, prompts, templates, examples, and scripts.
- Keep local-agent handoff templates bounded by repository, branch, file scope, forbidden actions, validation commands, and stop conditions.
- Avoid unverified adoption, user-count, or production-use claims.

## v0.2.x — Regression and portability

- Add fixture-based checks for unsafe local-execution claims.
- Add examples for Codex, Web ChatGPT, Claude Code, Gemini CLI, and Agy handoffs.
- Add compatibility notes for macOS, GitHub Actions, and local-only scripts.
- Add release checklist enforcement for public baseline changes.

## v0.3.x — Project template hardening

- Improve generated `AGENTS.md` and project command center templates.
- Add project-local public-safety checks for downstream app repositories.
- Add migration notes for installing the kit into existing projects.
- Add changelog and release cadence.

## Out of scope

- Do not claim official OpenAI, Anthropic, Google, or GitHub endorsement.
- Do not include private app source, customer data, employer data, local logs, or credentials.
- Do not optimize for synthetic stars, forks, or usage metrics.
