# Computer Use QA

Use only when explicitly requested.

## Prep

- Confirm simulator device and OS.
- Build/install app.
- Launch app and capture screenshot evidence.

## Matrix

- First launch.
- Navigation.
- Main workflows.
- Error/empty states.
- Accessibility labels.
- IAP purchase/restore if applicable.

## Exit

Create bug cards for failures. Do not claim absolute bug-free. Update NEXT_CODEX_PROMPT.
